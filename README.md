# GodotSharpPlus

[![Latest Stable Release (NuGet)](https://img.shields.io/nuget/v/GodotSharpPlus?style=plastic&logo=nuget&logoColor=FFFFFF&label=NuGet&labelColor=1C2B43&color=292929)](https://www.nuget.org/packages/GodotSharpPlus)

GodotSharpPlus is a collection of quality-of-life utilities for the Godot game engine, aiming to reduce boilerplate code
and enhance productivity. It includes extension members for Godot classes, plus source generator-provided attributes
applicable to Nodes.

Documentation is available on the [wiki](https://github.com/0xAlexisSys/GodotSharpPlus/wiki).

## Installation

Install via NuGet:

```bash
dotnet add package GodotSharpPlus
```
